## My contribution

- Design and develop a range of custom React components, including buttons, nested select menus, drop-down menus, badges, donut charts, and icons, improving the functionality and stability of the library.
- Migrate a React components library from JavaScript to TypeScript, improving the reliability and maintainability of the code.
- Introduced E2E testing practices to ensure the functionality and integrity of the application.
- Collaborated with product owners, designers, and other developers to deliver the product on an iterative basis.
- Implemented WCAG AA accessibility standards for a unified components library (@axe-core/react, static analysis, axe CLI).
