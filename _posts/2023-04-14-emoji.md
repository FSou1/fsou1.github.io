Can you guess, what will be the output `'рҹ‘©вҖҚрҹ‘©вҖҚрҹ‘§вҖҚрҹ‘§'.length // ?`?

It is `11`.

What about this one: `['вҳә', 'рҹҳҖ', '1пёҸвғЈ', 'рҹҮҰрҹҮЁ', 'рҹ‘ұвҖҚвҷӮпёҸ', 'рҹ‘©вҖҚвқӨпёҸвҖҚрҹ‘Ё', 'рҹ‘©вҖҚрҹ‘©вҖҚрҹ‘§вҖҚрҹ‘§'].map((emoji) => emoji.length)`?

Well, here it goes: `[1, 2, 3, 4, 5, 8, 11]`.

My favorite two examples look like marriage and divorce:

```
['рҹ‘Ё', 'вҖҚ', 'рҹ‘©'].reduce((cur, next) => cur += next) // рҹ‘ЁвҖҚрҹ‘©

[...'рҹ‘ЁвҖҚрҹ‘©вҖҚрҹ‘§']   // ['рҹ‘Ё', 'вҖҚ', 'рҹ‘©', 'вҖҚ', 'рҹ‘§']
```

I've group every emoji by it's length, so you don't have to (https://github.com/FSou1/EmojiGroupedByLength).
