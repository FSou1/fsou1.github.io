[Keyword Highlighter](https://chrome.google.com/webstore/detail/keywords-highlighter/lmaonnpepldfnbkbmhkkficpmjdlojnn) is a Chrome extension that highlights keywords on web pages, making it easy to find or skip information.

![image-left](/assets/images/projects/keywords-highlighter.png)

If you'd like to contribute to Keyword Highlighter, please feel free to [submit a pull request](https://github.com/FSou1/Keywords-Highlighter). We welcome any feedback or suggestions to improve the functionality of the extension.

Thank you for using Keyword Highlighter!
