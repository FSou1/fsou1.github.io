QuickReply is a Chrome extension that lets users save and use their common responses faster. It's built with JavaScript and Angular and you can see it in action in this video.

{% include video id="m9_ridH2EvE" provider="youtube" %}

This is one of the projects I'm using on a regular basis. Everytime I need to share a link to my Github or Email, I'm easily doing it with this extension.

Three main reasons why I've developed this project:

- Sometimes you might need to reply to a lot of people;
- And you don't always have time to personalize the answers;
- Or your answers are not always perfectly polite (but it's not that hard, is it?).
